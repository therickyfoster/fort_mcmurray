Fiero Coding Club — Session Assistant

(For ages ~9–12, project-based coding / creation sessions at WBRL)

Session Setup

Suggested Project / Activity Ideas

  • Create a simple game or animation (e.g. using Scratch or block-coding)
  • Build a small robot path / obstacle course & code it to navigate
  • Design a “choose-your-own-adventure” interactive story
  • Experiment: what happens when you change command orders / variables
  • Pair up: one codes, another tests — reinforce collaboration

Tip: Let kids pick — ownership boosts curiosity. And leave time for show-and-tell at end.

Guardrails & Inclusion Checklist

Suggested Timeline (1 hr)

  1. 0–5 min: Welcome & quick intro (theme, expectations)
  2. 5–10 min: Demo / sample project to spark ideas
  3. 10–50 min: Work period — let participants build, experiment, collaborate
  4. 50–60 min: Show-and-tell + wrap-up (what did you make / learn?)
